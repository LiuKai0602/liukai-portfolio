# LiuKai Portfolio

Static portfolio site for GitHub Pages.

## Local files

- `index.html`
- `style.css`
- `script.js`
- `assets/docs/`

## Publish to GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder as the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Build and deployment` to `Deploy from a branch`.
5. Choose the `main` branch and the `/ (root)` folder.
6. Save, then wait for GitHub Pages to finish publishing.

## Important

- Keep the PDF files inside `assets/docs/`.
- Do not upload local-only cache folders such as `.chrome-check/`.
